# นโยบายและกรอบการบริหารความเสี่ยง — บลจ. (เอกสารตัวอย่างสำหรับ RAG)

## 1. ขอบเขตและวัตถุประสงค์
เอกสารนี้กำหนดกรอบการบริหารความเสี่ยง (Risk Framework) ลิมิตความเสี่ยง และแนวทางควบคุม
สำหรับกองทุนภายใต้การจัดการ เพื่อให้การลงทุนอยู่ในระดับความเสี่ยงที่ยอมรับได้และเป็นไปตามเกณฑ์ ก.ล.ต.

## 2. ลิมิตความเสี่ยง (Risk Limits)

### 2.1 Value at Risk (VaR)
- กองทุนตราสารทุน (Equity): VaR รายวันที่ระดับความเชื่อมั่น 99% ต้องไม่เกิน 3.0%
- กองทุนตราสารหนี้ (Fixed Income): VaR รายวันที่ระดับความเชื่อมั่น 99% ต้องไม่เกิน 1.0%
- กองทุนผสม (Balanced): VaR รายวันที่ระดับความเชื่อมั่น 99% ต้องไม่เกิน 2.0%
- หากเกินลิมิต ผู้จัดการกองทุนต้องรายงานต่อคณะกรรมการบริหารความเสี่ยงภายใน 1 วันทำการ

### 2.2 Volatility (ความผันผวน)
- กองทุนตราสารทุน: ความผันผวนต่อปี (annualized volatility) ควรอยู่ในช่วง 12%–25%
- กองทุนตราสารหนี้: ความผันผวนต่อปีไม่ควรเกิน 6%
- หากความผันผวนเกินกรอบบน ให้ทบทวนการกระจายการลงทุน (diversification)

### 2.3 Maximum Drawdown
- กองทุนตราสารทุน: Max Drawdown ไม่ควรเกิน 25% หากเกินต้องจัดทำแผนลดความเสี่ยง
- กองทุนตราสารหนี้: Max Drawdown ไม่ควรเกิน 8%
- เมื่อ Drawdown แตะ 20% ให้แจ้งเตือนล่วงหน้า (early warning) ต่อหัวหน้าฝ่ายลงทุน

## 3. การวัดผลการดำเนินงาน (Performance Measurement)
- ใช้ดัชนี SET Total Return Index เป็น benchmark หลักของกองทุนหุ้นไทย
- Sharpe Ratio ที่ต่ำกว่า 0 ติดต่อกัน 2 ไตรมาส ต้องนำเข้าทบทวนกลยุทธ์
- ผลตอบแทนเทียบ benchmark (active return) รายงานเป็นรายเดือน

## 4. AI Governance และการควบคุมการใช้ AI
- ผลลัพธ์จากโมเดล AI/LLM ทุกชิ้นต้องตรวจสอบย้อนกลับได้ (traceable) และมีแหล่งข้อมูลกำกับ
- ตัวเลขทางการเงินทั้งหมดต้องคำนวณจากระบบที่ตรวจสอบได้ ห้ามให้ LLM สร้างตัวเลขเอง
- ข้อมูลที่ป้อนเข้าโมเดลต้องผ่านการตรวจสอบคุณภาพ (data quality check) ก่อนเสมอ
- ต้องมี human-in-the-loop สำหรับรายงานที่เสนอคณะกรรมการและลูกค้า
- ห้ามนำข้อมูลลูกค้าที่เป็นความลับเข้าสู่โมเดลภายนอกที่ไม่ได้รับอนุมัติ

## 5. ขั้นตอนการ Escalation เมื่อเกินลิมิต
1. ระบบตรวจพบการเกินลิมิต และแจ้งเตือนอัตโนมัติ (automated flag)
2. ผู้จัดการกองทุนตรวจสอบสาเหตุภายในวันเดียวกัน
3. รายงานต่อคณะกรรมการบริหารความเสี่ยงพร้อมแผนแก้ไข
4. ติดตามผลจนกลับเข้าสู่กรอบที่กำหนด
